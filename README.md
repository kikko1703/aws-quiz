# AWS Cloud Practitioner Quiz App

**Link al Progetto:** [Vai all'App](https://kikko1703.github.io/aws-quiz/)

## Descrizione
Web app interattiva per il self-testing sui servizi core di AWS (EC2, S3, IAM, VPC), sviluppata per consolidare le conoscenze richieste dalla certificazione Cloud Practitioner.

## Caratteristiche Tecniche
*   **Logica:** Sviluppata in JavaScript, con ottimizzazione del codice e del database domande effettuate tramite tecniche di Prompt Engineering.
*   **Deployment:** Hosting gestito tramite GitHub Pages con aggiornamento automatico dal repository.
*   **Contenuti:** Focus su infrastruttura, sicurezza e networking nel contesto AWS.
